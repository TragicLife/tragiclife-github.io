typescript-类型

2021-03-22

简介

TypeScript 是 JavaScript 的超集，它涵盖 ES6、ES7 等新语法特性。它同时也是一门静态弱类型语言，因为要兼容 JavaScript,因此 TypeScript 几乎不限制 JavaScript 中原有的隐式类型转换。

静态语言：在编译时期报错；动态语言：在运行时期报错

优势

  1. 规避大量低级错误，避免时间浪费，省时
  2. 减少多人协作项目的成本，大型项目友好，省力
  3. 良好代码提示，不用反复文件跳转或者翻文档，省心

基础类型

TypeScript 的原始类型包括: boolean、number、string、void、undefined、null、symbol、bigint。

  • boolean
  • number
  • string
  • void, 只有 null 和 undefined 可以赋值给 void
