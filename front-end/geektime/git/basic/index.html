安装 git

Git 的特点

  • 最优的存储能力
  • 非凡的性能
  • 开源的
  • 很容易做备份
  • 支持离线操作
  • 很容易定制工作流程

配置 user 信息

config 的三个作用域

  • --local：只对某个仓库有效
  • --global：对当前用户所以仓库有效
  • --system：对系统所有登录的用户有效

创建 git 仓库

  • 把已有的项目代码纳入 git 管理

  • 新建的项目直接用 git 管理

    • cd 代码目录
    • git init project
    • cd project

展示最近 4 条日志

git log -n4

查看所有分支的

git log --all
